# Summary

This is a remix of the great  Bear-X-Ends to fit non Prusa TR8 nuts. These brass nuts are a little bit smaller in diameter, so that the mounting holes are closer by each other.

The x-end-motor comes in 3 versions. All Fusion360 files are uploaded, so you can modify these files for your needs.
The X-End with the bearing is designed for 625zz or 625rs bearings.

Print settings are all like the original bear-upgrades.

Have Fun!
